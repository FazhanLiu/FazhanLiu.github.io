# FazhanLiu.github.io

Personal website published with GitHub Pages.

Live site: https://fazhanliu.github.io/

## Overview

This is a clean personal homepage built with plain HTML, CSS, and JavaScript. It includes sections for profile, skills, projects, experience, and contact information.

## Local Preview

```bash
python3 -m http.server 5173
```

Then open:

```text
http://localhost:5173
```

## Customize

- Edit `index.html` to update name, role, profile text, projects, and contact links.
- Edit `styles.css` to adjust the visual style.
- Edit `script.js` to tune the subtle background animation.
